# Maa Bhawani Agencies — Premium Showroom Website

A modern, premium, and fully responsive showroom website for **Maa Bhawani Agencies**, an authorized distributor of kitchen sinks, sanitary ware, tiles, bathroom fittings, and plumbing solutions.

The website delivers a luxury digital showroom experience with elegant layouts, premium visuals, smooth animations, and a clean, modern user interface.

---

## ✨ Features

- 🎨 Premium luxury showroom UI
- 📱 Fully responsive design
- ⚡ Fast and optimized performance
- 🧩 Modular React component architecture
- 🎭 Smooth animations and micro-interactions
- 🪟 Glassmorphism-inspired UI
- 🖼️ Product showcase and gallery
- 🏢 Brand-focused presentation

## 🚀 Getting Started

### Clone the Repository

```bash
git clone <repository-url>
```

### Navigate to the Project

```bash
cd maa-bhawani-agencies
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
